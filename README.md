# Group Scholar Outcome Atlas

Outcome Atlas is a lightweight evidence ledger for tracking scholarship program outcomes, confidence signals, and narrative context. It runs as a static web app with local storage to keep outcome notes and proof links close to the team.

## Highlights
- Outcome portfolio with category, status, owner, and metric signals
- Evidence timeline and confidence pulse
- Local storage persistence plus JSON export

## Run locally
Open `index.html` in a browser.
